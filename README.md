# Tetris Clone

Classic tetris game

![gameplay](docs/gameplay.gif)

## Controls

Move block horizontally => Left and Right Arrows, 'A' and 'D' keys.

Rotate block => Up arrow, 'W' key

Move block vertically => Down arrow, 'S' key
